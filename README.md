# Advent of Code 2025
## Picotto Francesco - Solutions in Java

### Prerequisites:
- JDK: jdk-23
- Build Tool: Maven

## Install
Copy this in your terminal

```
git clone https://github.com/francesco-picotto/adventofcode2025.git
```
Then type
```
cd adventofcode2025
```
## Project Structure
```
src/
 ├── main/
 │    ├── java/       # Codice sorgente Java
 │    └── resources/  # File di configurazione (application.properties, log4j)
 └── test/
      └── java/       # Test unitari e di integrazione
```
